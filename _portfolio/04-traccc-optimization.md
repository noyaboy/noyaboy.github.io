## Overview

High-performance GPU optimization for the traccc particle tracking pipeline at the High-Luminosity LHC. This project focuses on accelerating real-time trajectory reconstruction algorithms through low-level CUDA kernel tuning and memory access optimizations, aiming to maximize throughput on heterogeneous computing platforms.
